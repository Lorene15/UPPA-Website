Project Description
This project was developed as part of the "Web Technologies" course.
The main goal of the project is to present famous paintings grouped by different art periods, such as:
- Impressionism  
- Cubism  

The website provides a simple and user-friendly navigation and includes multimedia elements to enhance the user experience.

Website Structure
- Home: Header section with SVG and project title  
- Paintings: Display of paintings grouped by art periods, including images and descriptions  
- Contact: Contact form and location displayed using Google Maps  
- Footer: Copyright and references
